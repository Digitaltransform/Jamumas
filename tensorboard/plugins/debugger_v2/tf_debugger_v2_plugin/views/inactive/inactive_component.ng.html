Debugger V2 is inactive because no data is available.
To use the debugger,
  1. Add the following line to the beginning of your program:
    tf.debugging.experimental.enable_dump_debug_info( logdir )
  2. Re-run the program.

The log directory must contain TensorFlow Debugger (V2) data. tf.debugging.experimental.enable_dump_debug_info(logdir) will collect tensor data, graph structures, the associated stack traces, and source code to the specificed directory logdir as the instrumented TensorFlow program executes.
